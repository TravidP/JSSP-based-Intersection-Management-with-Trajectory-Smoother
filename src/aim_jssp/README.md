# `aim_jssp` Package Plan

This package will contain the future implementation of the autonomous intersection management scheduler.

Planned subsystems:

- `graphs/`: operation graph construction and feature extraction.
- `env/`: abstract JSSP/AIM environment and simulator adapters.
- `models/`: relation-aware GAT policy networks.
- `training/`: rollout, policy-gradient training, evaluation, and logging.

This folder is currently documentation-only.
